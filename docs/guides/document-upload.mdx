# 文档上传

## 支持的文件格式

| 格式 | 扩展名 | MIME 类型 |
|------|--------|-----------|
| PDF | `.pdf` | `application/pdf` |
| Word | `.doc`, `.docx` | `application/msword`, `application/vnd.openxmlformats-officedocument.wordprocessingml.document` |
| 纯文本 | `.txt` | `text/plain` |
| Markdown | `.md` | `text/markdown` |

## 上传流程

1. 进入知识库详情页
2. 点击"上传文档"按钮
3. 选择文件（可多选）
4. 等待上传完成，文档状态变为 `PENDING`
5. 后台自动进行解析、分块、向量化
6. 状态变为 `COMPLETED` 后即可用于问答

## 文档状态

| 状态 | 说明 | 操作 |
|------|------|------|
| `PENDING` | 已创建，等待后台处理 | 等待 |
| `PROCESSING` | 正在解析、分块、向量化 | 等待 |
| `COMPLETED` | 处理完成，可用于问答 | 删除 |
| `FAILED` | 处理失败 | 重试或删除 |

## 查重机制

系统使用 SHA-256 内容哈希进行查重：

- 同一文件改个名字再传 → 正确拒绝
- 不同文件恰好同名 → 正确允许
- 文件内容修改后再传 → 允许（哈希变了）

## 重试失败文档

如果文档处理失败：

1. 在文档列表中找到状态为 `FAILED` 的文档
2. 点击"重试"按钮
3. 系统会清理残留数据，重新提交处理任务
