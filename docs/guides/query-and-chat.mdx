# 问答与聊天

## 发起问答

1. 进入知识库详情页
2. 点击"开始问答"进入聊天页面
3. 在输入框中输入问题
4. 点击发送或按回车键

## 流式输出

系统使用 SSE（Server-Sent Events）技术实现打字机效果：

- 后端生成一个字，前端立刻显示一个字
- 不需要等待全部生成完成
- 支持实时引用溯源

## 引用溯源

回答中的 `[1]`、`[2]` 等标记表示引用来源：

- 点击引用编号可展开查看原始文档片段
- 每个引用包含文档名称、片段内容和相似度分数
- `validation` 事件验证引用编号的合法性

## 新建会话

点击"新对话"按钮可以创建新的聊天会话：

- 每个知识库可以有多个会话
- 会话之间历史消息隔离
- 旧会话可以随时切换回来继续

## 消息历史

- 会话历史自动保存
- 支持滚动加载更多历史消息
- 旧历史达到阈值后自动压缩为摘要

## 注意事项

- 问答只能基于当前知识库的文档内容
- 如果文档中没有相关信息，系统会明确说明
- 生成过程中可以中断，但已生成的内容不会保存
