# 知识库管理

## 创建知识库

1. 登录后进入知识库列表页
2. 点击"新建知识库"按钮
3. 输入知识库名称和描述
4. 点击确认创建

## 知识库列表

列表页展示当前用户有权限访问的所有知识库：

- `USER` 只能看到自己创建的知识库
- `ADMIN` 可以看到所有知识库

## 删除知识库

<Danger>
  删除知识库会同时清理该知识库下的所有文档元数据、Milvus 向量和 Mongo 关键词索引。此操作不可恢复！
</Danger>

删除流程：

1. 在知识库列表页点击删除按钮
2. 确认删除对话框
3. 后端异步清理相关数据
4. 清理完成后知识库从列表消失

## 知识库详情

进入知识库详情页可以：

- 查看已上传的文档列表
- 上传新文档
- 进入聊天问答页面
- 查看文档处理状态
