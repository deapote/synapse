# 管理员操作

## 用户管理

`ADMIN` 角色可以：

- 查看所有用户列表
- 创建新用户
- 修改用户角色
- 启用/禁用用户账号

### 创建用户

需要提供：

- 用户名（唯一）
- 显示名
- 密码（至少 8 位）
- 角色（`USER` 或 `ADMIN`）

### 修改用户角色

可以为用户分配一个或多个角色。角色的变更立即生效，用户下次操作时使用新的权限。

### 启停用户

禁用用户后：

- 用户无法登录
- 已颁发的 token 失效
- 不影响已创建的知识库和文档

## 角色权限管理

`ADMIN` 可以修改角色的权限配置：

| 角色 | 默认权限 |
|------|---------|
| `ADMIN` | KB_READ、KB_WRITE、KB_DELETE、AUTH_ADMIN |
| `USER` | KB_READ、KB_WRITE |

<Warning>
  修改角色权限会影响所有拥有该角色的用户，请谨慎操作。
</Warning>
