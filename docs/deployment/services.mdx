# 外部服务配置

## MongoDB

### Docker 启动

```bash
docker run -d --name synapse-mongo \
  -p 27017:27017 \
  -e MONGO_INITDB_ROOT_USERNAME=admin \
  -e MONGO_INITDB_ROOT_PASSWORD=password \
  -v mongo-data:/data/db \
  mongo:6.0
```

### 连接配置

```yaml
spring:
  data:
    mongodb:
      uri: mongodb://admin:password@localhost:27017/synapse
```

## Milvus

### Standalone 模式

```bash
# 下载并启动
curl -sfL https://raw.githubusercontent.com/milvus-io/milvus/master/scripts/standalone_embed.sh -o standalone_embed.sh
bash standalone_embed.sh start
```

### Docker Compose

```yaml
version: '3.5'
services:
  milvus:
    image: milvusdb/milvus:v2.3.0
    ports:
      - "19530:19530"
      - "9091:9091"
    volumes:
      - milvus-data:/var/lib/milvus
volumes:
  milvus-data:
```

## Ollama

### 安装

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.com/install.sh | sh
```

### 启动服务

```bash
ollama serve
```

### 拉取模型

```bash
# 对话模型
ollama pull qwen2.5:7b

# Embedding 模型
ollama pull gme-qwen2-vl:2b
```

### 自定义模型配置

如需使用其他模型，修改 `application.yaml`：

```yaml
synapse:
  ollama:
    base-url: http://localhost:11434
    chat-model: your-chat-model
    embedding-model: your-embedding-model
    embedding-dimension: 1536
```

<Warning>
  切换 Embedding 模型时，需要重新创建 Milvus collection 以匹配新的向量维度。已有数据需要重新摄入。
</Warning>
