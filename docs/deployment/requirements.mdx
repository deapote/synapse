# 环境要求

## 软件依赖

| 依赖 | 版本要求 | 说明 |
|------|---------|------|
| Java | 21+ | 后端运行环境 |
| Maven | 3.9+ | 构建工具 |
| Node.js | 18+ | 前端构建 |
| MongoDB | 6.0+ | 元数据存储 |
| Milvus | 2.3+ | 向量存储 |
| Ollama | 最新 | LLM 和 Embedding 服务 |

## 推荐硬件配置

### 开发环境

| 资源 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 4 核 | 8 核 |
| 内存 | 8 GB | 16 GB |
| 磁盘 | 20 GB SSD | 50 GB SSD |

### 生产环境

| 资源 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 8 核 | 16 核 |
| 内存 | 16 GB | 32 GB |
| 磁盘 | 100 GB SSD | 500 GB SSD |

<Tip>
  Ollama 运行大模型需要较多内存。qwen2.5:7b 模型约需 8GB 内存，建议为 Ollama 单独分配足够资源。
</Tip>

## 端口占用

| 服务 | 端口 | 说明 |
|------|------|------|
| 后端 API | 8082 | Spring Boot 应用 |
| 前端 DevServer | 5173 | Vite 开发服务器 |
| MongoDB | 27017 | 元数据存储 |
| Milvus | 19530 | 向量存储 gRPC |
| Milvus Web | 9091 | Milvus Web UI |
| Ollama | 11434 | LLM 服务 |
