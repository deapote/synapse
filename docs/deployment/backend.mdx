# 后端部署

## 开发环境启动

```bash
# 编译并运行
cd synapse-kb-bootstrap
mvn spring-boot:run
```

后端默认运行在 `http://localhost:8082`。

## 生产环境打包

```bash
# 根目录执行
cd /Users/admin/codeProject/synapse
mvn clean package -DskipTests

# 产物在 synapse-kb-bootstrap/target/
java -jar synapse-kb-bootstrap/target/synapse-kb-bootstrap-*.jar
```

## 配置文件

生产环境配置集中在 `synapse-kb-bootstrap/src/main/resources/application.yaml`：

```yaml
server:
  port: 8082

spring:
  data:
    mongodb:
      uri: mongodb://admin:password@localhost:27017/synapse

synapse:
  milvus:
    host: localhost
    port: 19530
    collection: synapse_document_chunks
  ollama:
    base-url: http://localhost:11434
    chat-model: qwen2.5:7b
    embedding-model: gme-qwen2-vl:2b
  cors:
    allowed-origins: https://your-frontend-domain.com
  upload:
    max-file-bytes: 10485760  # 10MB
    allowed-extensions: pdf,doc,docx,txt,md
  web:
    max-page-size: 100
```

## 虚拟线程配置

文档摄入和 RAG 检索使用独立 Executor，默认使用虚拟线程：

```yaml
synapse:
  ingestion:
    executor:
      type: virtual    # 或 fixed
      pool-size: 4     # fixed 类型时生效
  retrieval:
    executor:
      type: virtual
```
