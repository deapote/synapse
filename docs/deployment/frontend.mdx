# 前端部署

## 开发环境

```bash
cd synapse-frontend
npm install
npm run dev
```

前端默认运行在 `http://localhost:5173`。

## 生产构建

```bash
cd synapse-frontend
npm install
npm run build
```

构建产物在 `synapse-frontend/dist/` 目录下，可以部署到任何静态文件服务器。

## 环境变量

创建 `.env.production`：

```
VITE_API_BASE_URL=https://your-api-domain.com
```

## Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-frontend-domain.com;

    location / {
        root /path/to/synapse-frontend/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://localhost:8082;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## Docker 部署

```dockerfile
FROM node:20-alpine AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
```
