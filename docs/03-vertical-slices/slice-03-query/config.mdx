# 问答检索 —— Config 层

## RAG 相关配置

```yaml
synapse:
  rag:
    top-k: 5                    # 返回引用数
    vector-candidate-k: 20      # 向量候选数
    keyword-candidate-k: 20     # 关键词候选数
    vector-weight: 0.65         # 向量检索权重
    keyword-weight: 0.35        # 关键词检索权重
    query-rewrite:
      enabled: true             # 启用 Query 改写
      similarity-threshold: 0.8 # 改写质量门禁阈值
    prompt-template: |
      基于以下参考资料回答用户问题：
      {context}
      用户问题：{question}
      请基于以上参考资料回答，如果资料不足以回答，请准确说明。
```

- [ ] 为什么向量权重（0.65）比关键词权重（0.35）高？
