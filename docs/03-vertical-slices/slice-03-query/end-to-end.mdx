# 问答检索 —— 完整链路

```mermaid
sequenceDiagram
    participant Browser
    participant StreamingQueryController
    participant QueryUseCase
    participant SessionRepo
    participant QueryRewrite
    participant Embedding
    participant VectorSearch
    participant KeywordSearch
    participant LLM

    Browser->>StreamingQueryController: POST /query/stream
    Note over Browser,StreamingQueryController: {query, sessionId}

    StreamingQueryController->>QueryUseCase: prepare(query)

    QueryUseCase->>SessionRepo: getOrCreateSession()
    QueryUseCase->>QueryUseCase: appendMessage(USER, query)

    QueryUseCase->>QueryRewrite: rewrite(query.text())
    QueryRewrite->>QueryRewrite: embed(original) vs embed(rewritten)
    QueryRewrite-->>QueryUseCase: rewrittenQuery (or original if low similarity)

    QueryUseCase->>Embedding: embed(query.text())
    Embedding-->>QueryUseCase: queryEmbedding

    par 并行检索
        QueryUseCase->>VectorSearch: search(kbId, queryEmbedding, topK)
        VectorSearch-->>QueryUseCase: vectorResults
    and
        QueryUseCase->>KeywordSearch: search(kbId, query.text())
        KeywordSearch-->>QueryUseCase: keywordResults
    end

    QueryUseCase->>QueryUseCase: mergeAndRerank(vectorResults, keywordResults)
    QueryUseCase->>QueryUseCase: buildPrompt(mergedResults, query)

    QueryUseCase-->>StreamingQueryController: RagContext(prompt, references)

    StreamingQueryController->>LLM: generateStream(prompt)
    LLM-->>StreamingQueryController: Stream<token>

    loop 逐字推送
        StreamingQueryController-->>Browser: SSE: token
    end

    StreamingQueryController-->>Browser: SSE: references
    StreamingQueryController-->>Browser: SSE: validation
    StreamingQueryController-->>Browser: SSE: complete
```

## SSE 事件协议

```
event: session
data: {"sessionId": "xxx"}

event: token
data: {"token": "这"}

event: token
data: {"token": "个"}

event: token
data: {"token": "产"}

...

event: references
data: {"references": [{"id": 1, "text": "...", "source": "..."}]}

event: validation
data: {"trusted": true, "usedSourceIds": [1, 2]}

event: complete
data: {}
```

| 事件类型 | 时机 | 内容 |
|---------|------|------|
| `session` | 开始时 | 会话 ID |
| `token` | 每个字 | 生成的文本片段 |
| `references` | 生成完后 | 引用来源列表 |
| `validation` | 最后 | 引用校验结果 |
| `complete` | 结束时 | 空对象 |
| `error` | 出错时 | 错误信息 |

| 7 | `KnowledgeBaseApplicationService.java:prepare()` | `CompletableFuture` 并行检索 |

| 14 | `StreamingQueryController.java` | 包装成 SSE 事件逐字推送 |
