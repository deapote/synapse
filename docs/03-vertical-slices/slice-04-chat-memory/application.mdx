# 聊天记忆 —— Application 层

## 记忆组装

文件：`synapse-kb-application/.../port/service/KnowledgeBaseApplicationService.java`

```java
private String appendMemoryContext(ChatSession session) {
    if (!chatMemoryEnabled) {
        return "";
    }

    StringBuilder memory = new StringBuilder();

    // 1. 添加摘要
    if (session.getSummary() != null && !session.getSummary().isBlank()) {
        memory.append("<conversation_summary>\n")
              .append(session.getSummary())
              .append("\n</conversation_summary>\n\n");
    }

    // 2. 添加最近消息
    List<ChatMessage> recentMessages = chatMessageRepository
        .findRecentBySessionIdBeforeOrEqual(session.getId(), session.getMessageCount(), recentMessageLimit);

    if (!recentMessages.isEmpty()) {
        memory.append("<recent_dialogue>\n");
        for (ChatMessage msg : recentMessages) {
            memory.append(msg.role() == ChatRole.USER ? "用户: " : "助手: ")
                  .append(msg.content())
                  .append("\n");
        }
        memory.append("\n</recent_dialogue>\n");
    }

    return memory.toString();
}
```

## 摘要触发

```java
private void summarizeIfNeeded(ChatSession session) {
    long unsummarizedCount = session.getMessageCount() - session.getSummarizedUntilSequence();
    if (unsummarizedCount < summaryTriggerMessageCount) {
        return;  // 未达到触发阈值
    }

    // 获取未摘要的消息
    List<ChatMessage> messages = chatMessageRepository
        .findBySessionIdAndSequenceBetween(
            session.getId(),
            session.getSummarizedUntilSequence(),
            session.getMessageCount()
        );

    // 生成新摘要
    String newSummary = chatMemorySummarizerPort.summarize(
        session.getSummary(), messages, maxSummaryChars
    );

    session.updateSummary(newSummary, session.getMessageCount());
    chatSessionRepository.save(session);
}
```
