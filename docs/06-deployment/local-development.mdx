# 部署 —— 本地开发环境

## 环境要求

| 组件 | 版本 | 用途 |
|------|------|------|
| Java | 21 | 后端运行 |
| Maven | 3.9+ | 构建 |
| Node.js | 18+ | 前端运行 |
| MongoDB | 6.0+ | 元数据存储 |
| Milvus | 2.3+ | 向量存储 |
| Ollama | 最新 | 本地 LLM |

## Docker Compose 一键启动依赖

```yaml
# docker-compose.dev.yml
version: '3.8'
services:
  mongodb:
    image: mongo:6
    ports:
      - "27017:27017"
    volumes:
      - mongo_data:/data/db

  milvus-standalone:
    image: milvusdb/milvus:v2.3.3
    ports:
      - "19530:19530"
    volumes:
      - milvus_data:/var/lib/milvus

  milvus-attu:
    image: zilliz/attu:v2.3.5
    ports:
      - "8000:3000"
    environment:
      - MILVUS_URL=milvus-standalone:19530

volumes:
  mongo_data:
  milvus_data:
```

```bash
docker-compose -f docker-compose.dev.yml up -d
```

## 配置文件说明

`synapse-kb-bootstrap/src/main/resources/application.yaml`

```yaml
spring:
  data:
    mongodb:
      uri: mongodb://localhost:27017/synapse

synapse:
  llm:
    base-url: http://localhost:11434
    model-name: qwen2.5:7b

  embedding:
    base-url: http://localhost:11434
    model-name: gme-Qwen2-VL-2B-Instruct-GGUF:Q8_0
    dimension: 1536

  milvus:
    host: localhost
    port: 19530
    collection: document_chunks

  ingestion:
    max-file-size: 52428800        # 50MB
    allowed-extensions: pdf,docx,txt,md
    chunk-max-segment-size: 1000
    chunk-overlap-ratio: 0.15

  rag:
    top-k: 5
    vector-weight: 0.65
    keyword-weight: 0.35
    query-rewrite-enabled: true
    query-rewrite-similarity-threshold: 0.8

  chat-memory:
    enabled: true
    recent-message-limit: 8
    summary-trigger-message-count: 12
    max-summary-chars: 1500
```

## 启动后端

```bash
# 方式一：IDE 中运行 SynapseApplication.main()
# 方式二：命令行
mvn clean install -DskipTests
mvn spring-boot:run -pl synapse-kb-bootstrap
```

## 启动前端

```bash
cd synapse-frontend
npm install
npm run dev
```

访问 `http://localhost:5173`

## 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | synapse-admin | ADMIN |

**⚠️ 生产环境必须修改默认密码。**
