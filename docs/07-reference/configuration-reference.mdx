# 参考 —— 配置参考

## 完整配置示例

```yaml
spring:
  data:
    mongodb:
      uri: mongodb://localhost:27017/synapse

sa-token:
  token-name: satoken
  timeout: 2592000
  activity-timeout: -1

synapse:
  web:
    max-page-size: 100

  llm:
    base-url: http://localhost:11434
    model-name: qwen2.5:7b

  embedding:
    base-url: http://localhost:11434
    model-name: gme-Qwen2-VL-2B-Instruct-GGUF:Q8_0
    dimension: 1536

  milvus:
    host: localhost
    port: 19530
    collection: document_chunks

  upload:
    max-file-bytes: 52428800
    allowed-extensions: pdf,doc,docx,txt,md
    allowed-content-types: application/pdf,application/msword,application/vnd.openxmlformats-officedocument.wordprocessingml.document,text/plain,text/markdown

  ingestion:
    chunk-max-segment-size: 1000
    chunk-overlap-ratio: 0.15
    chunk-min-overlap-size: 50
    chunk-max-overlap-size: 200
    max-attempts: 3
    lease-duration: PT5M

  rag:
    top-k: 5
    vector-candidate-k: 50
    keyword-candidate-k: 50
    vector-weight: 0.65
    keyword-weight: 0.35
    query-rewrite-enabled: true
    query-rewrite-similarity-threshold: 0.8
    prompt-template: |
      以下是检索到的知识库片段：
      %s

      请根据以上片段回答用户问题。如果片段中没有相关信息，请诚实说明。
      用户问题：%s

  chat-memory:
    enabled: true
    recent-message-limit: 8
    summary-trigger-message-count: 12
    max-summary-chars: 1500
```

## 配置项说明

### Web

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.web.max-page-size` | 100 | 分页最大每页数量 | 否 |

### LLM

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.llm.base-url` | http://localhost:11434 | Ollama 服务地址 | 否 |
| `synapse.llm.model-name` | qwen2.5:7b | 对话模型名称 | 否 |

### Embedding

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.embedding.base-url` | http://localhost:11434 | Ollama 服务地址 | 否 |
| `synapse.embedding.model-name` | gme-Qwen2-VL-2B-Instruct-GGUF:Q8_0 | Embedding 模型 | 否 |
| `synapse.embedding.dimension` | 1536 | 向量维度 | 否 |

### Milvus

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.milvus.host` | localhost | Milvus 主机 | 否 |
| `synapse.milvus.port` | 19530 | Milvus 端口 | 否 |
| `synapse.milvus.collection` | document_chunks | Collection 名称 | 否 |

### 上传

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.upload.max-file-bytes` | 52428800 | 最大文件大小（字节），默认 50MB | 否 |
| `synapse.upload.allowed-extensions` | pdf,doc,docx,txt,md | 允许的文件扩展名 | 否 |
| `synapse.upload.allowed-content-types` | ... | 允许的 MIME 类型 | 否 |

### 摄入

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.ingestion.chunk-max-segment-size` | 1000 | 分块最大字符数 | 否 |
| `synapse.ingestion.chunk-overlap-ratio` | 0.15 | 分块重叠比例 | 否 |
| `synapse.ingestion.chunk-min-overlap-size` | 50 | 重叠最小字符数 | 否 |
| `synapse.ingestion.chunk-max-overlap-size` | 200 | 重叠最大字符数 | 否 |
| `synapse.ingestion.max-attempts` | 3 | 摄入失败最大重试次数 | 否 |
| `synapse.ingestion.lease-duration` | PT5M | 任务租约时长（ISO 8601） | 否 |

### RAG

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.rag.top-k` | 5 | 最终返回的引用数量 | 否 |
| `synapse.rag.vector-candidate-k` | 50 | 向量检索候选集大小 | 否 |
| `synapse.rag.keyword-candidate-k` | 50 | 关键词检索候选集大小 | 否 |
| `synapse.rag.vector-weight` | 0.65 | 向量检索融合权重 | 否 |
| `synapse.rag.keyword-weight` | 0.35 | 关键词检索融合权重 | 否 |
| `synapse.rag.query-rewrite-enabled` | true | 是否启用 Query 改写 | 否 |
| `synapse.rag.query-rewrite-similarity-threshold` | 0.8 | Query 改写相似度阈值 | 否 |

### 聊天记忆

| 配置项 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| `synapse.chat-memory.enabled` | true | 是否启用聊天记忆 | 否 |
| `synapse.chat-memory.recent-message-limit` | 8 | 保留最近消息数 | 否 |
| `synapse.chat-memory.summary-trigger-message-count` | 12 | 触发摘要的消息阈值 | 否 |
| `synapse.chat-memory.max-summary-chars` | 1500 | 摘要最大字符数 | 否 |
