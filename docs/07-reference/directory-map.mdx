# 参考 —— 项目目录速查

## 后端模块

```
synapse/
├── synapse-shared/                    # 共享内核
│   └── src/main/java/com/synapse/shared/
│       └── exception/                 # DomainException
│
├── synapse-auth-domain/               # 认证领域层 [切片1 - Domain]
│   └── .../auth/model/                # UserAccount, RoleDefinition, UserId...
│   └── .../auth/repository/           # 仓储接口
│
├── synapse-auth-application/          # 认证应用层 [切片1 - Application]
│   └── .../auth/port/
│       ├── in/                        # AuthenticationUseCase, UserAdminUseCase
│       ├── out/                       # LoginSessionPort, PasswordHasherPort
│       └── service/                   # AuthApplicationService
│
├── synapse-auth-adapter/              # 认证适配器层 [切片1 - Adapter]
│   └── .../auth/adapter/in/web/       # AuthController, UserAdminController
│   └── .../auth/adapter/out/          # SaTokenLoginSessionAdapter, BCryptPasswordHasherAdapter...
│
├── synapse-auth-config/               # 认证配置 [切片1 - Config]
│   └── AuthBeanConfig, AuthDataInitializer, SaTokenSecurityConfig
│
├── synapse-kb-domain/                 # 知识库领域层 [切片2/3/4 - Domain]
│   └── .../kb/model/                  # KnowledgeBase, Document, ChatSession...
│   └── .../kb/repository/             # 仓储接口
│   └── .../kb/service/                # RecursiveChunkingStrategy [横向深入]
│
├── synapse-kb-application/            # 知识库应用层 [切片2/3/4 - Application]
│   └── .../kb/port/
│       ├── in/                        # 各种 UseCase
│       ├── out/                       # VectorStorePort, EmbeddingPort...
│       └── service/                   # KnowledgeBaseApplicationService
│
├── synapse-kb-adapter/                # 知识库适配器层 [切片2/3/4 - Adapter]
│   └── .../kb/adapter/in/web/         # DocumentController, StreamingQueryController...
│   └── .../kb/adapter/out/            # MilvusVectorStoreAdapter, OllamaEmbeddingAdapter...
│
├── synapse-kb-config/                 # 知识库配置 [切片2/3/4 - Config]
│   └── KnowledgeBaseBeanConfig
│
└── synapse-kb-bootstrap/              # 启动入口
    └── SynapseApplication.java
    └── resources/application.yaml     # 所有配置 [配置参考]
```

## 前端模块

```
synapse-frontend/
├── src/
│   ├── api/                           # API 客户端 [前端架构]
│   │   ├── client.ts                  # axios 配置
│   │   └── query.ts                   # SSE 手动解析
│   ├── stores/                        # Pinia Store [前端架构]
│   │   └── chat.ts                    # 核心：竞态防护、Token 缓冲
│   ├── composables/                   # 可复用逻辑
│   │   └── usePolling.ts              # 轮询 Hook
│   ├── views/                         # 页面
│   ├── components/                    # 通用组件
│   └── router/                        # 路由配置
```
