# API 参考 —— 聊天

## GET /api/knowledge-bases/{kbId}/chat/sessions/current

获取当前知识库的会话（不存在则自动创建）。

### 请求

```bash
curl http://localhost:8080/api/knowledge-bases/kb-uuid/chat/sessions/current \
  -H "satoken: xxx"
```

### 响应

**成功 (200)**：

```json
{
  "id": "session-uuid",
  "knowledgeBaseId": "kb-uuid",
  "title": "新对话",
  "summary": null,
  "messageCount": 0,
  "createdAt": "2024-01-15T08:30:00Z",
  "updatedAt": "2024-01-15T08:30:00Z"
}
```

### 权限

`KB_READ`

---

## POST /api/knowledge-bases/{kbId}/chat/sessions

创建新会话。

### 请求

```bash
curl -X POST http://localhost:8080/api/knowledge-bases/kb-uuid/chat/sessions \
  -H "satoken: xxx"
```

### 响应

**成功 (200)**：会话信息（同上）。

### 权限

`KB_READ`

---

## GET /api/chat/sessions/{sessionId}/messages

获取会话消息列表。

### 请求

```bash
curl "http://localhost:8080/api/chat/sessions/session-uuid/messages?page=0&size=50" \
  -H "satoken: xxx"
```

**查询参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 0 | 页码 |
| size | int | 50 | 每页数量，最大 100 |

### 响应

**成功 (200)**：

```json
[
  {
    "id": "msg-uuid",
    "sessionId": "session-uuid",
    "role": "user",
    "content": "如何安装产品？",
    "references": [],
    "sequence": 1,
    "createdAt": "2024-01-15T08:30:00Z"
  },
  {
    "id": "msg-uuid-2",
    "sessionId": "session-uuid",
    "role": "assistant",
    "content": "根据产品手册...",
    "references": [
      {
        "sourceId": 1,
        "documentId": "doc-uuid",
        "documentName": "安装指南.pdf",
        "chunkText": "第一步：下载安装包...",
        "score": 0.9234,
        "startPosition": 0,
        "endPosition": 200,
        "used": true
      }
    ],
    "sequence": 2,
    "createdAt": "2024-01-15T08:30:05Z"
  }
]
```

### 权限

`KB_READ`

### 说明

- 只能查询自己创建的会话消息
- 按 `sequence` 升序排列
