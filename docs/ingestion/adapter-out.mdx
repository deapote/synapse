# 文档上传与摄入 —— Adapter-Out 层

## ApacheTikaDocumentParserAdapter

文件：`synapse-kb-adapter/.../out/parser/ApacheTikaDocumentParserAdapter.java`

```java
@Component
public class ApacheTikaDocumentParserAdapter implements DocumentParserPort {
    private final AutoDetectParser parser = new AutoDetectParser();

    @Override
    public String parse(InputStream inputStream, String fileName) {
        try (InputStream is = inputStream) {
            byte[] bytes = is.readNBytes(maxParserInputBytes);
            rejectDangerousPdfForms(bytes, fileName);

            Metadata metadata = new Metadata();
            metadata.set(TikaCoreProperties.RESOURCE_NAME_KEY, fileName);
            BodyContentHandler handler = new BodyContentHandler(maxParsedChars);
            parser.parse(new ByteArrayInputStream(bytes), handler, metadata, parseContext());
            return handler.toString();
        } catch (Exception e) {
            throw new DomainException("文档解析失败: " + fileName, e);
        }
    }

    private ParseContext parseContext() {
        PDFParserConfig pdfConfig = new PDFParserConfig();
        pdfConfig.setExtractAcroFormContent(false);   // 禁用 AcroForm
        pdfConfig.setIfXFAExtractOnlyXFA(false);      // 禁用 XFA
        pdfConfig.setExtractInlineImages(false);      // 不提取图片
        pdfConfig.setExtractActions(false);           // 不提取动作
        pdfConfig.setOcrStrategy(PDFParserConfig.OCR_STRATEGY.NO_OCR);

        ParseContext context = new ParseContext();
        context.set(PDFParserConfig.class, pdfConfig);
        return context;
    }
}
```

### 安全设计

PDF 可能包含恶意内容（JavaScript、表单动作等）。Tika 的 `PDFParserConfig` 禁用了这些功能，只提取纯文本内容。

## OllamaEmbeddingAdapter

文件：`synapse-kb-adapter/.../out/embedding/OllamaEmbeddingAdapter.java`

```java
@Component
public class OllamaEmbeddingAdapter implements EmbeddingPort {
    private final OllamaEmbeddingModel embeddingModel;
    private final Timer embeddingTimer;

    public OllamaEmbeddingAdapter(@Value("${ollama.base-url}") String baseUrl,
                                   @Value("${ollama.embedding-model}") String modelName,
                                   MeterRegistry meterRegistry) {
        this.embeddingModel = OllamaEmbeddingModel.builder()
            .baseUrl(baseUrl)
            .modelName(modelName)
            .timeout(Duration.ofSeconds(120))
            .build();
        this.embeddingTimer = Timer.builder("synapse.embedding")
            .register(meterRegistry);
    }

    @Override
    @CircuitBreaker(name = "ollamaEmbedding")
    @Retry(name = "ollamaEmbedding")
    @Bulkhead(name = "ollamaEmbedding")
    public float[] embed(String text) {
        return embeddingTimer.record(() -> {
            return embeddingModel.embed(text).content().vector();
        });
    }

    @Override
    @CircuitBreaker(name = "ollamaEmbedding")
    @Retry(name = "ollamaEmbedding")
    @Bulkhead(name = "ollamaEmbedding")
    public List<float[]> embed(List<String> texts) {
        return embeddingTimer.record(() -> {
            List<TextSegment> segments = texts.stream()
                .map(TextSegment::from).toList();
            return embeddingModel.embedAll(segments).content()
                .stream().map(Embedding::vector).toList();
        });
    }
}
```

### 弹性设计

- **`@CircuitBreaker`**：Ollama 连续失败时快速失败，避免拖垮系统
- **`@Retry`**：临时失败时自动重试
- **`@Bulkhead`**：限制并发调用数，防止资源耗尽
- **`Timer`**：监控 Embedding 耗时

## MilvusVectorStoreAdapter

文件：`synapse-kb-adapter/.../out/vector/MilvusVectorStoreAdapter.java`

```java
@Component
public class MilvusVectorStoreAdapter implements VectorStorePort {
    @Override
    public void store(KnowledgeBaseId kbId, DocumentId docId, String docName,
                      List<DocumentChunk> chunks, List<float[]> embeddings) {
        List<JsonObject> rows = new ArrayList<>();
        for (int i = 0; i < chunks.size(); i++) {
            DocumentChunk chunk = chunks.get(i);
            float[] embedding = embeddings.get(i);
            JsonObject row = new JsonObject();
            row.addProperty("id", docId.value() + "-" + i);
            row.addProperty("knowledgeBaseId", kbId.value());
            row.addProperty("documentId", docId.value());
            row.addProperty("chunkIndex", i);
            row.addProperty("chunkText", chunk.text());
            row.add("vector", toJsonArray(embedding));
            rows.add(row);
        }
        milvusClient.insert(collectionName, rows);
    }

    @Override
    public List<ChunkReference> search(KnowledgeBaseId kbId, float[] queryEmbedding, int topK) {
        // 构建查询条件：必须匹配 knowledgeBaseId
        QueryWrapper wrapper = new QueryWrapper();
        wrapper.eq("knowledgeBaseId", kbId.value());

        SearchParam param = SearchParam.newBuilder()
            .withCollectionName(collectionName)
            .withVectors(List.of(queryEmbedding))
            .withTopK(topK)
            .withMetricType(MetricType.COSINE)
            .withExpr("knowledgeBaseId == '" + kbId.value() + "'")
            .build();

        return milvusClient.search(param).getResults().stream()
            .map(this::toChunkReference).toList();
    }
}
```

### 安全边界：禁止跨知识库查询

```java
// 上传时：向量带上了知识库 ID
vectorStorePort.store(kbId, documentId, chunks, embeddings);

// 查询时：只搜索指定知识库
vectorStorePort.search(kbId, queryEmbedding, topK);
```

如果不传 `kbId`，用户在知识库 B 提问时可能检索到知识库 A 的文档，造成数据泄露。

## GridFsDocumentContentStoreAdapter

文件：`synapse-kb-adapter/.../out/persistence/GridFsDocumentContentStoreAdapter.java`

```java
@Component
public class GridFsDocumentContentStoreAdapter implements DocumentContentStorePort {
    private final GridFsTemplate gridFsTemplate;

    @Override
    public String store(KnowledgeBaseId kbId, DocumentId docId, String fileName,
                        String contentType, InputStream content) {
        Document metadata = new Document()
            .append("knowledgeBaseId", kbId.value())
            .append("documentId", docId.value());
        ObjectId objectId = gridFsTemplate.store(content, fileName, contentType, metadata);
        return objectId.toHexString();
    }
}
```

GridFS 是 MongoDB 的文件存储系统，适合存储大文件（超过 16MB 的文档）。
