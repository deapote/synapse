# 文档上传与摄入 —— Application 层

## 入站端口

文件：`synapse-kb-application/.../port/in/IngestDocumentUseCase.java`

```java
public interface IngestDocumentUseCase {
    DocumentId ingest(IngestDocumentCommand command);

    record IngestDocumentCommand(
        KnowledgeBaseId knowledgeBaseId,
        String fileName,
        String contentType,
        long fileSize,
        String contentHash,
        InputStream content,
        DocumentMetadata metadata
    ) {
        public IngestDocumentCommand {
            if (metadata == null) {
                metadata = new DocumentMetadata();
            }
        }
    }
}
```

`IngestDocumentCommand` 现在包含 `DocumentMetadata`，所有时效字段通过上传接口传入。

## ingest() 方法 —— 同步上传

文件：`synapse-kb-application/.../port/service/KnowledgeBaseApplicationService.java`

```java
@Override
public DocumentId ingest(IngestDocumentCommand command) {
    // 1. 权限检查
    KnowledgeBase kb = requireKnowledgeBase(command.knowledgeBaseId());
    accessControlPort.checkKnowledgeBaseAccess(kb, "KB_WRITE");

    // 2. 重复内容检查
    List<Document> existingDocs = documentRepository
        .findByKnowledgeBaseIdAndContentHash(command.knowledgeBaseId(), command.contentHash());
    for (Document existing : existingDocs) {
        if (existing.getStatus() == PENDING || existing.getStatus() == PROCESSING
                || existing.getStatus() == COMPLETED) {
            throw new DomainException("此知识库已存在相同内容的文档");
        }
        if (existing.getStatus() == FAILED) {
            cleanupFailedDocument(existing);
        }
    }

    // 3. 校验 supersedesDocumentId（如果存在）
    DocumentMetadata metadata = command.metadata();
    if (metadata.supersedesDocumentId() != null && !metadata.supersedesDocumentId().isBlank()) {
        DocumentId oldId = new DocumentId(metadata.supersedesDocumentId());
        Document oldDoc = documentRepository.findById(oldId)
            .orElseThrow(() -> new DomainException("被替代文档不存在"));
        if (!oldDoc.getKnowledgeBaseId().equals(command.knowledgeBaseId())) {
            throw new DomainException("被替代文档必须属于同一知识库");
        }
        if (oldDoc.getLifecycleStatus() != DocumentLifecycleStatus.ACTIVE) {
            throw new DomainException("被替代文档必须是 ACTIVE 状态");
        }
    }

    // 4. 创建文档（携带 metadata）
    Document document = Document.create(
        command.knowledgeBaseId(), command.fileName(),
        command.contentType(), command.fileSize(), command.contentHash(), metadata
    );

    // 5. 存储原始内容到 GridFS
    String contentObjectId = documentContentStorePort.store(
        command.knowledgeBaseId(), document.getId(),
        document.getFileName(), document.getFileType(), command.content()
    );
    document.attachContentObject(contentObjectId);

    // 6. 保存文档（失败则回滚 GridFS）
    try {
        document = documentRepository.save(document);
    } catch (RuntimeException e) {
        documentContentStorePort.delete(contentObjectId);
        throw e;
    }

    // 7. 创建摄入任务（失败则回滚文档和 GridFS）
    try {
        ingestionJobRepository.save(
            IngestionJob.create(document.getId(), document.getKnowledgeBaseId(), contentObjectId)
        );
    } catch (RuntimeException e) {
        documentRepository.deleteById(document.getId());
        documentContentStorePort.delete(contentObjectId);
        throw e;
    }

    return document.getId();
}
```

### 版本替代校验

上传时传入 `supersedesDocumentId` 会做前置校验：
- 被替代文档必须存在
- 必须属于同一知识库
- 必须是 `ACTIVE` 状态

**注意**：此时**不**立即修改旧文档。旧文档的替代发生在后台处理成功之后。

### 补偿逻辑分析

```
存 GridFS → 存 Document → 存 IngestionJob
     ↑___________↑___________↑
     失败时回滚上一步
```

这是一个**链式补偿**：每步失败时回滚之前所有步骤。

## processDocument() —— 异步处理

```java
private void processDocument(Document document, String contentObjectId) {
    try {
        if (document.getStatus() == PENDING) {
            document.transitionTo(PROCESSING);
            documentRepository.save(document);
        }

        // 1. 解析文档
        List<DocumentChunk> chunks = parseDocumentChunks(document, contentObjectId);

        // 2. 存储索引（携带 metadata）
        storeDocumentIndexes(document, chunks);

        document.setChunkCount(chunks.size());
        document.transitionTo(COMPLETED);

        // 3. 新文档摄入成功后，再执行版本替代
        if (document.getSupersedesDocumentId() != null
                && !document.getSupersedesDocumentId().isBlank()) {
            supersedeOldDocument(document);
        }

    } catch (Exception e) {
        cleanupDocumentIndexesQuietly(document);
        throw new DomainException("文档处理失败: " + safeFailureReason(e), e);
    } finally {
        documentRepository.save(document);
    }
}
```

### 版本替代时机

```java
private void supersedeOldDocument(Document newDocument) {
    DocumentId oldId = new DocumentId(newDocument.getSupersedesDocumentId());
    Document oldDoc = documentRepository.findById(oldId).orElse(null);
    if (oldDoc == null) {
        log.warn("被替代文档不存在，跳过替代");
        return;
    }

    oldDoc.supersedeBy(newDocument, newDocument.getEffectiveFrom());
    documentRepository.save(oldDoc);
}
```

**关键设计**：旧文档只在新文档**摄入成功并 COMPLETED 后**才标记 `SUPERSEDED`。如果新文档摄入失败，旧文档保持 `ACTIVE`，系统不会失去有效资料。

### parseDocumentChunks()

```java
private List<DocumentChunk> parseDocumentChunks(Document document, String contentObjectId) {
    InputStream content = documentContentStorePort.open(contentObjectId);
    String text = documentParserPort.parse(content, document.getFileName());
    List<DocumentChunk> chunks = recursiveChunkingStrategy.split(text);
    if (chunks.isEmpty()) {
        throw new DomainException("文档未解析出有效文本");
    }
    return chunks;
}
```

### storeDocumentIndexes()

```java
private void storeDocumentIndexes(Document document, List<DocumentChunk> chunks) {
    // 1. Embedding
    List<float[]> embeddings = embeddingPort.embed(
        chunks.stream().map(DocumentChunk::text).toList()
    );

    // 2. 向量存储（Milvus v3）
    DocumentMetadata metadata = toMetadata(document);
    vectorStorePort.store(
        document.getKnowledgeBaseId(), document.getId(),
        document.getFileName(), chunks, embeddings, metadata
    );

    // 3. 关键词索引（MongoDB）
    chunkSearchIndexPort.store(
        document.getKnowledgeBaseId(), document.getId(),
        document.getFileName(), chunks, metadata
    );
}
```

**双写索引**：向量索引和关键词索引必须同时成功，且都携带 `DocumentMetadata`。如果其中一个失败，`cleanupDocumentIndexesQuietly()` 会清理已写入的数据。

## 失败处理

```java
private void handleIngestionFailure(IngestionJob job, Document document, Exception e) {
    String reason = safeFailureReason(e);
    if (job.getAttempts() >= ingestionMaxAttempts) {
        tryTransitionToFailed(document, reason);
        documentRepository.save(document);
        job.markFailed(reason);
    } else {
        job.markRetrying(reason, Instant.now().plus(backoffForAttempt(job.getAttempts())));
    }
    ingestionJobRepository.save(job);
}
```

**退避策略**：
- 第1次失败后：30 秒后重试
- 第2次失败后：2 分钟后重试

## v1 限制

```java
@Override
public Document updateMetadata(DocumentId id, DocumentMetadata metadata) {
    throw new DomainException("v1 不支持在线修改已完成文档的元数据，请通过重新摄入新版本完成时效变更");
}

@Override
public void supersede(DocumentId oldDocumentId, DocumentId newDocumentId, LocalDate effectiveTo) {
    throw new DomainException("v1 不支持在线替代文档，请通过上传新版本并指定 supersedesDocumentId 完成时效变更");
}
```

v1 不支持在线修改已完成文档的 metadata，也不支持手动在线 supersede。正确的资料时效变更必须通过重新上传新版本完成。
