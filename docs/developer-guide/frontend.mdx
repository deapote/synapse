# 前端架构

`synapse-frontend` 是基于 Vue 3 + Vite + Pinia 的单页应用，提供知识库管理、文档上传、问答聊天等功能。

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.4 | 前端框架 |
| Vite | 5+ | 构建工具 |
| Pinia | - | 状态管理 |
| Vue Router | 4 | 路由管理 |
| Axios | - | HTTP 客户端 |
| TypeScript | - | 类型安全 |

## Axios 客户端

```typescript
// api/client.ts
const client = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || 'http://localhost:8082',
  headers: { 'Content-Type': 'application/json' }
});

// 请求拦截器：自动注入 token
client.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers['synapse-token'] = token;
  }
  return config;
});

// 响应拦截器：401 跳转登录
client.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) {
      localStorage.removeItem('token');
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);
```

## SSE 流式问答

```typescript
// stores/chat.ts
async function streamQuery(kbId: string, query: string) {
  const response = await fetch(
    `${API_BASE}/api/knowledge-bases/${kbId}/query/stream`,
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'synapse-token': getToken()
      },
      body: JSON.stringify({ query })
    }
  );

  const reader = response.body?.getReader();
  const decoder = new TextDecoder();

  while (reader) {
    const { done, value } = await reader.read();
    if (done) break;

    const text = decoder.decode(value);
    const events = parseSSEEvents(text);

    for (const event of events) {
      switch (event.event) {
        case 'token':
          appendToken(event.data.token);
          break;
        case 'references':
          setReferences(event.data.references);
          break;
        case 'validation':
          setValidation(event.data);
          break;
        case 'complete':
          finishStream();
          return;
        case 'error':
          setError(event.data.message);
          return;
      }
    }
  }
}
```

## 状态管理

使用 Pinia 管理全局状态：

- **auth**：登录状态、用户信息、token
- **knowledgeBase**：知识库列表、当前知识库
- **chat**：会话列表、当前会话、消息列表、流式输出状态
- **document**：文档列表、上传进度

## 关键设计

1. **自动轮询**：文档列表页使用 `usePolling` 轮询文档状态，实时更新处理进度
2. **消息隔离**：聊天消息按 `sessionId` 存储，切换会话时自动加载历史消息
3. **引用展示**：答案中的 `[1]` 引用可点击，展开显示原始文档片段
