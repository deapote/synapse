# 参考 —— 术语表

## DDD 术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 领域驱动设计 | Domain-Driven Design | 以业务领域为核心组织代码的软件开发方法 |
| 实体 | Entity | 有唯一标识、可变状态的对象 |
| 值对象 | Value Object | 无标识、不可变、按内容相等的对象 |
| 聚合根 | Aggregate Root | 一组相关对象的入口，外部只能通过它访问内部对象 |
| 领域服务 | Domain Service | 不属于任何实体的业务逻辑 |
| 仓储 | Repository | 聚合根的持久化抽象，只定义接口 |
| 充血模型 | Rich Domain Model | 数据和行为封装在一起的模型 |
| 贫血模型 | Anemic Domain Model | 只有 getter/setter 的数据容器 |
| 有界上下文 | Bounded Context | 业务边界内的统一领域模型 |
| 防腐层 | Anti-Corruption Layer | 隔离外部系统，防止外部模型污染内部领域 |

## RAG 术语

| 术语 | 英文 | 说明 |
|------|------|------|
| RAG | Retrieval-Augmented Generation | 检索增强生成，用外部知识库辅助 LLM 回答 |
| Embedding | Embedding | 将文本转换为高维向量的过程 |
| 向量检索 | Vector Search | 基于向量相似度查找相关内容 |
| 关键词检索 | Keyword Search | 基于关键词匹配的文本检索 |
| 混合检索 | Hybrid Retrieval | 向量检索 + 关键词检索融合 |
| 分块 | Chunking | 将长文本切分为适合处理的小段 |
| 提示词 | Prompt | 给 LLM 的输入文本 |
| SSE | Server-Sent Events | 服务器向客户端推送事件的单向流协议 |
| 召回 | Recall | 检索阶段，从库中找出候选内容 |
| 重排 | Rerank | 对召回结果重新排序，选出最佳内容 |

## 资料时效性术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 资料时效性 | Temporal Validity | 文档/法规在特定时间点是否有效的业务属性 |
| asOfDate | As-of Date | 查询时指定的"时间点"，检索只返回在该时间点有效的资料 |
| effectiveFrom | Effective From | 资料生效日期（含） |
| effectiveTo | Effective To | 资料排他结束日期；`effectiveTo=2025-01-01` 表示 2025-01-01 当天起无效 |
| 排他结束日 | Exclusive End Date | `effectiveTo` 的语义：到达该日期即失效，不包含该日期 |
| canonicalKey | Canonical Key | 规范标识，用于同一资料的多版本关联和去重 |
| supersede | Supersede | 用新版本替代旧版本；旧文档标记 `SUPERSEDED` |
| lifecycleStatus | Lifecycle Status | 业务时效状态：`ACTIVE`（生效中）、`SUPERSEDED`（已被替代）、`RETIRED`（已废止） |
| RetrievalFilter | Retrieval Filter | 检索过滤条件，含必填 `asOfDate` 和可选 `sourceType`/`jurisdiction` |
| 时效硬过滤 | Temporal Hard Filter | 在检索阶段直接排除过期资料，不是重排加权 |
| 兜底过滤 | Fallback Filter | Application 层用 Mongo `Document` 真实状态做的二次时效校验 |

## Spring 术语

| 术语 | 英文 | 说明 |
|------|------|------|
| WebFlux | Spring WebFlux | Spring 的响应式 Web 框架 |
| Mono | Mono | Reactor 中异步返回 0 或 1 个值的类型 |
| Flux | Flux | Reactor 中异步返回 0 到 N 个值的类型 |
| 端口 | Port | 六边形架构中的接口，定义系统与外部交互的契约 |
| 适配器 | Adapter | 端口的具体实现，连接外部技术 |
| 入站端口 | Driving Port | 外部调用系统的接口（如 UseCase） |
| 出站端口 | Driven Port | 系统调用外部的接口（如 SPI） |
| Bean | Bean | Spring 管理的对象实例 |
| 配置类 | @Configuration | 定义 Bean 的 Spring 配置类 |
