# API 参考 —— 认证

## POST /api/auth/login

用户登录。

### 请求

```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"synapse-admin"}'
```

**请求体**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名 |
| password | string | 是 | 密码 |

### 响应

**成功 (200)**：

```json
{
  "id": "user-uuid",
  "username": "admin",
  "displayName": "管理员",
  "roles": ["ADMIN"],
  "permissions": ["KB_READ", "KB_WRITE", "KB_DELETE", "AUTH_ADMIN"],
  "tokenName": "satoken",
  "tokenValue": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
}
```

**失败 (400/401)**：

```json
{
  "message": "用户名或密码错误"
}
```

### 权限

无需登录（白名单接口）。

## POST /api/auth/logout

用户登出。

### 请求

```bash
curl -X POST http://localhost:8080/api/auth/logout \
  -H "satoken: xxx"
```

### 响应

**成功 (200)**：无响应体。

### 权限

需要登录。

---

## GET /api/auth/me

获取当前登录用户信息。

### 请求

```bash
curl http://localhost:8080/api/auth/me \
  -H "satoken: xxx"
```

### 响应

**成功 (200)**：

```json
{
  "id": "user-uuid",
  "username": "admin",
  "displayName": "管理员",
  "roles": ["ADMIN"],
  "permissions": ["KB_READ", "KB_WRITE", "KB_DELETE", "AUTH_ADMIN"],
  "enabled": true,
  "createdAt": "2024-01-15T08:30:00Z"
}
```

### 权限

需要登录。
