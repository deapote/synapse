# API 参考 —— 管理

所有接口需要 `AUTH_ADMIN` 权限。

## POST /api/admin/users

创建用户。

### 请求

```bash
curl -X POST http://localhost:8080/api/admin/users \
  -H "Content-Type: application/json" \
  -H "satoken: xxx" \
  -d '{
    "username": "zhangsan",
    "displayName": "张三",
    "password": "password123",
    "roles": ["USER"]
  }'
```

**请求体**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名，1-64字符 |
| displayName | string | 否 | 显示名称 |
| password | string | 是 | 明文密码，会自动哈希 |
| roles | string[] | 否 | 角色列表，默认 USER |

### 响应

**成功 (200)**：

```json
{
  "id": "user-uuid",
  "username": "zhangsan",
  "displayName": "张三",
  "roles": ["USER"],
  "permissions": ["KB_READ", "KB_WRITE"],
  "enabled": true,
  "createdAt": "2024-01-15T08:30:00Z"
}
```

## GET /api/admin/users

分页查询用户列表。

### 请求

```bash
curl "http://localhost:8080/api/admin/users?page=0&size=20" \
  -H "satoken: xxx"
```

**查询参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 0 | 页码 |
| size | int | 20 | 每页数量，最大 100 |

### 响应

**成功 (200)**：用户列表数组。

---

## PUT /api/admin/users/{id}/roles

修改用户角色。

### 请求

```bash
curl -X PUT http://localhost:8080/api/admin/users/user-uuid/roles \
  -H "Content-Type: application/json" \
  -H "satoken: xxx" \
  -d '{"roles": ["USER", "ADMIN"]}'
```

---

## PUT /api/admin/users/{id}/enabled

启用/禁用用户。

### 请求

```bash
curl -X PUT http://localhost:8080/api/admin/users/user-uuid/enabled \
  -H "Content-Type: application/json" \
  -H "satoken: xxx" \
  -d '{"enabled": false}'
```

---

## GET /api/admin/roles

查询所有角色及其权限。

### 请求

```bash
curl http://localhost:8080/api/admin/roles \
  -H "satoken: xxx"
```

### 响应

**成功 (200)**：

```json
[
  {
    "name": "ADMIN",
    "permissions": ["KB_READ", "KB_WRITE", "KB_DELETE", "AUTH_ADMIN"]
  },
  {
    "name": "USER",
    "permissions": ["KB_READ", "KB_WRITE"]
  }
]
```

---

## PUT /api/admin/roles/{roleName}/permissions

修改角色权限。

### 请求

```bash
curl -X PUT http://localhost:8080/api/admin/roles/USER/permissions \
  -H "Content-Type: application/json" \
  -H "satoken: xxx" \
  -d '{"permissions": ["KB_READ"]}'
```
