# 第一次使用

## 1. 登录

打开 `http://localhost:5173`，用默认管理员账号登录：

- 用户名：`admin`
- 密码：`ChangeMe123!`

登录成功后进入知识库列表页面。

## 2. 创建知识库

点击"新建知识库"，输入：

- 名称：`产品手册`
- 描述：`存放公司产品相关文档`

确认后知识库出现在列表中，点击进入详情页。

## 3. 上传文档

在知识库详情页点击"上传文档"，选择 PDF/Word/TXT/Markdown 文件。

文档状态流转：

- **PENDING** — 刚上传，等待处理
- **PROCESSING** — 正在解析、分块、向量化
- **COMPLETED** — 处理完成，可检索
- **FAILED** — 处理失败，可重试

状态刷新周期取决于文档大小和机器性能。

## 4. 问答

文档状态为 COMPLETED 后，点击"开始问答"进入聊天页面。输入与文档内容相关的问题，系统返回基于检索片段生成的答案。

交互特征：

- **流式输出**：SSE 逐字推送，无需等待全文生成
- **引用溯源**：回答中的 `[1]`、`[2]` 可点击跳转原始片段

## 5. 多轮对话

连续提问时，系统自动携带前文上下文。历史超过阈值后自动压缩为摘要，避免 prompt 超限。

## 6. 多用户隔离（管理员功能）

进入"用户管理"创建新用户：

- 用户名：`testuser`
- 显示名：`测试用户`
- 密码：`Test123456!`
- 角色：`USER`

切换为该用户登录后：

- 只能访问自己创建的知识库
- 无法访问其他用户（包括 admin）的知识库
- 无法进入"用户管理"页面（无 `AUTH_ADMIN` 权限）
